PIC32MZW1 Wi-Fi® MCU Module Family
The PIC32MZ "W1" Wi-Fi® Module family is a series of Wi-Fi modules with integrated MCU and advanced security designed specifically for IoT applications. It is a secure, powerful, and robust IoT platform used by developers worldwide to create millions of smart things around you.

If You Already Have a PIC32MZ "W1" Wi-Fi® MCU IoT Board:

  1. Connect the PIC32MZ "W1" Wi-Fi® MCU IoT board to your laptop using a standard micro-USB cable
  2. Create WIFI.CFG file using WiFi Config
  3. Copy the newly created WIFI.CFG file to WFI32-IoT MSD emulated on WFI32-IoT board
  4. Reboot the board! Now you are connected to WiFi network

Trust&Go Wi-Fi MCU module

WFI32E01PC is a class leading Wi-Fi MCU module, combining high-performance 32-bit MCU, robust Wi-Fi, and advanced Trust&Go security. WFI32E01PC contains a highly efficient MCU Core with numerous high-precision peripherals, such as dual 12-bit ADCs, integrating three systems elements into one!

Security

The WFI32E01PC Wi-Fi module features Trust&Go technology, which provides advanced hardware security and authentication. It is pre-provisioned for three leading cloud platforms; AWS, GCP, and Azure.
